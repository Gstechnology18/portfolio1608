3° trimestre

Habilidades: H1, H2, H3 e H4.

O objetivo dessa atividade foi criar uma aplicação web simples utilizando o framework Express para gerenciar rotas e servir páginas HTML. Além disso, a atividade visou aprender a estilizar essas páginas com CSS, modificando cores de fundo e texto, para proporcionar uma experiência visual agradável e acessível ao usuário.
